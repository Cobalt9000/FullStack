Courses for structure/fundamentals

HTML

HTML is used to structure and format content on the web, facilitating the creation of web pages by defining elements and their relationships.


CSS

CSS is utilized to style and design HTML documents, enhancing the visual presentation of web pages through control of layout, colors, and typography.


JavaScript

JavaScript is employed for creating interactive and dynamic features on websites, enhancing user experience through functionalities like form validation, animations, and real-time updates.


Framework

React.JS

React is a JavaScript library for building user interfaces, facilitating the creation of interactive and efficient UI components in web applications.


Django

Django is a high-level web framework for building robust and scalable web applications using the Python programming language.


Node.JS

Node.js is a JavaScript runtime environment that allows developers to execute server-side JavaScript, enabling scalable and efficient web application development.


Database

MySQL

MySQL is an open-source relational database management system (RDBMS) used for efficiently storing, managing, and retrieving structured data in web applications.


MongoDB

MongoDB is a NoSQL, document-oriented database designed for flexibility and scalability, commonly used in web development for storing and managing diverse, unstructured data.


FireBase

Firebase is a comprehensive mobile and web development platform by Google, offering real-time database, authentication, hosting, and other cloud services to simplify and accelerate app development.
